🐦 Flappy Bird Game (Frontend)

A simple and interactive Flappy Bird Game Clone built using HTML, CSS, and JavaScript, inspired by the popular arcade game.

This project demonstrates game development basics, including animations, collision detection, and score tracking in a browser environment.

🌐 Overview

This game allows users to:

Control a bird using keyboard or click actions

Navigate through obstacles (pipes)

Avoid collisions to survive longer

Score points based on performance

Flappy Bird-style games are commonly used to learn game loops, physics, and collision detection in web development

✨ Features

🎮 Classic Flappy Bird gameplay

🐦 Bird movement with user interaction

🧱 Dynamic obstacle generation

💥 Collision detection system

🧮 Score tracking

🎨 Simple and engaging UI

⚡ Smooth animations

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Concepts: Game Logic, Animations, Event Handling
